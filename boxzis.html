Box-Sizing Comparison

Ambas cajas tienen width: 200px, padding: 20px y border: 10px

content-box (por defecto)

Width declarado: 200px

Padding: 20px

Border: 10px

Tamaño total: 260px

Cálculo:

200px (width)
+ 40px (padding)
+ 20px (border)
= 260px

border-box (recomendado)

Width declarado: 200px

Padding: 20px

Border: 10px

Tamaño total: 200px

Cálculo:

200px total
padding y border incluidos

= 200px

¿Por qué usar border-box?

Predecible

El ancho real es el que declaras.

Responsive

Evita desbordes en layouts flex y grid.

Mantenimiento

No recalculas tamaños al cambiar padding.

Estándar

Usado por frameworks modernos.

Best Practice

*,
*::before,
*::after {
    box-sizing: border-box;
}

Inclúyelo siempre al inicio de tus proyectos.
